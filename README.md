# Notebooks

Bookkeeping some notebooks

## Preperation

### Requirements

* [pyenv](https://github.com/pyenv/pyenv)

### Setup

```sh
$ pyenv virtualenv 3.7.2 notebooks
$ pyenv activate notebooks
$ pip install --upgrade pip
$ pip install -r requirements.txt
```
